Aplikasi Pemesanan Brownies "Brunee"

Aplikasi ini dibuat sebagai tugas Ujian Tengah Semester mata kuliah **Pengembangan Aplikasi Mobile**.  
"Brunee" adalah aplikasi pemesanan brownies dengan tampilan menarik dan navigasi sederhana.  
Pengguna dapat melakukan **registrasi, login, memilih varian brownies, mengisi alamat, dan melihat halaman konfirmasi pesanan**.

---

## Fitur Aplikasi

1. **Splash Screen**  
   Menampilkan logo dan nama aplikasi saat pertama kali dibuka.

2. **Sign Up, Login & Register**  
   Pengguna dapat membuat akun dan masuk dengan username serta password.

3. **Home Page (Pilihan Brownies)**  
   Menampilkan berbagai varian brownies seperti:
   - Brownies Classic  
   - Brownies Nuts  
   - Brownies Cheese  
   - Brownies Espresso  
   - Brownies Oreo  

   Setiap varian dilengkapi dengan gambar dan deskripsi.

4. **Order Page**  
   Pengguna dapat memilih varian brownies yang diinginkan.

5. **Alamat Page**  
   Form untuk mengisi alamat pengiriman.

6. **Selesai Page**  
   Halaman konfirmasi pesanan telah berhasil.

---

## Teknologi yang Digunakan

- **Bahasa Pemrograman:** Kotlin  
- **Layout:** XML  
- **IDE:** Android Studio  
- **Versi Android SDK:** Minimal SDK 24  
- **Navigasi antar Activity:** Intent  
- **Struktur Tampilan:** LinearLayout, RelativeLayout, ScrollView
